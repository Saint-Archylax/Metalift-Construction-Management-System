# METALIFT: Construction Management System (CMS)

## Overview

METALIFT: Construction Management System (CMS) is a web-based platform designed to streamline and centralize construction project operations. The system integrates project management, employee management, finance tracking, inventory monitoring, and material management into a single platform, helping reduce manual processes, improve data accuracy, and support better decision-making.

## Key Features

### Project Management Subsystem (PMS)

* Create and manage construction projects
* Assign employees to projects
* Monitor project milestones and progress
* Submit and validate progress reports
* Generate project status reports

### Team Management Subsystem (TMS)

* Manage employee records
* Track attendance and project assignments
* Submit and review progress reports
* Handle profile update requests
* View employee documents and payroll information

### Finance Management Subsystem (FMS)

* Manage payroll records
* Track project expenses
* Record manual financial transactions
* Generate financial summaries and reports
* Monitor project-related costs

### Inventory Management Subsystem (IMS)

* Monitor material stock levels
* Record stock-in and stock-out transactions
* Track material usage per project
* Generate low-stock notifications
* Maintain inventory movement history

### Material Management Subsystem (MMS)

* Manage material records and specifications
* Maintain supplier information
* Standardize material data
* Track material history and updates
* Integrate material information with inventory management

## System Objectives

* Centralize construction management processes
* Improve project monitoring and reporting
* Automate attendance and payroll tracking
* Enhance inventory and material control
* Provide accurate financial records and reports
* Reduce manual errors and data redundancy

## User Roles

* **Project Manager** – Manages projects, employees, inventory, materials, and reports.
* **Finance User** – Handles payroll, expenses, and financial records.
* **Employee** – Submits progress reports, views project assignments, and accesses personal records.
